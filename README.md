## Objetivo del Sistema
El objetivo de este software es brindar una herramienta sencilla y automatizada a los ciudadanos para crear contraseñas seguras y aleatorias en milisegundos. El sistema busca romper los patrones de error humano (como usar cumpleaños o secuencias lógicas) mediante la selección matemática y ciega de caracteres, para evitar el riesgo de hackeos.
#

## Descripción de Funcionalidades
1. **Generación Aleatoria:** Crea claves combinando letras mayúsculas, minúsculas, números y símbolos usando el módulo matemático `random`.
2. **Validación de Seguridad:** El programa obliga al usuario a ingresar una longitud mínima de 8 caracteres. Si pone menos, el sistema muestra una advertencia.
3. **Módulo de Concienciación:** Incluye una opción en el menú para desplegar consejos prácticos sobre ciberseguridad ciudadana.
4. **Ciclo Continuo:** El sistema se mantiene activo mediante un menú repetitivo hasta que el usuario decide salir explícitamente.
